## Getting Started

Installation is straightforward:

```bash
# Install CADET OPT
git clone https://github.com/NLR-CADET/cadet-opt.git
cd cadet-opt
pip install -e .

# Install CADET Stochastic (optional)
git clone https://github.com/NLR-CADET/cadet-stochastic.git
cd cadet-stochastic
pip install -e .[gdmloader]

# Install examples (optional)
git clone https://github.com/NLR-CADET/cadet-examples.git
cd cadet-examples
pip install -e ".[dev]"
```

Visit our [Getting Started Guide](/cadet_webpage/pages/getting-started) for detailed instructions.
